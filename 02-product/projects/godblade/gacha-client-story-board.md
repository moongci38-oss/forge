개요
마을 화면 하단 ui 개편 및 짝 맞추기 시스템(동일 아이템이 나오면 당첨)

참고 문서 : 장비뽑기연출기획서

1. 마을 아래 항목 구성 변경
  좌측 : 상점, 가방, 거래소(신규), 우편함, 아바타, 조력자
  우측 : 뽑기(신규), 도전(신규)
  제외 : 스킬, 보석함, 길드, 연금술, 제작
  기획서 4 페이지
  거래소는 버튼만 배치(임의 이미지 처리)
  
  
2. 뽑기 연출
 기획서 9~14 페이지
 뽑기 상점 진입(기존 상점 ui 사용) -> 뽑기버튼 클릭 -> 상자 개봉 연출(패 깔리는 연출까지) -> 카드 뒤집기(수동, 모두 열기) -> 획득 결과
 상자 개봉 연출 안될시 사용할 에셋 : https://assetstore.unity.com/packages/3d/props/treasure-chest-with-effects-263013
 상자 개봉 연출 : 기획서 8 페이지

3. 뽑기 씬 구성
 카드 "10장"
 ui 하단에 "한번 더(숨김)", "모두 열기" 버튼 배치
  모두 열기는 자동 진행이다.
 한번 더는 비활성화 상태(숨김)였다가 카드를 모두 열었을 때 활성화(보여지게)으로 변경

- 수동 조작 연출
1) 카드가 뒤집어 지면서 해당 카드에 배치된 아이템이 등장한다.
2) 해당 카드 장비 정보는 기존에 갓검의 가챠에 사용되는 ui와 동일하다.
3) 카드 뒤집는 과정에서 "이미 뒤집어진 카드들 중에서 동일 아이템이 나올 경우" "짝 맞추기 연출"을 진행한다.

- 자동 조작 연출
수동과 같은 연출이나, 카드가 한장씩 자동으로 오픈되면 된다.
10장의 카드 중 "오픈 순서"는 "랜덤으로 하나씩" 오픈하면 된다.

- 짝 맞추기 연출
1단계 : 동일 아이템 카드 주변으로 "밝아 지면서 번개 이펙트" (번개 및 글로우 이펙트는 장비의 별(성)에 따라 색상이 틀리게 적용)
2단계 : 동일 아이템 카드가 한 바퀴 돌면서 강조
3단계 : 동일 아이템 카드는 "이펙트"를 통해서 강조

- 짝 맞추기 색상
  1~3성 : RGB 255,255,255
  4~6성 : RGB 146,208,80
  7~9성 : RGB 0,176,240
  10~11성 : RGB 204,0,0

결과창
- 모두 열린후 획득 아이템이 있을시 "장비획득 팝업창"
   없을시는 "중앙에 연기 이펙트가 터진 이후, "다음 기회에" 글자를 출력 후 기존 화면으로 돌아간다.
   기획서 15 페이지
   
- 모두 열린 후 하단 버튼 처리
 "한번 더" 버튼 보이게 처리
 "모두 열기" 빠지고, "상점 가기" 버튼으로 교체

- "모두 열기"는 뽑기버튼 클릭을 한번 진행하면 되고, "상점 가기"는 상점으로 이동하면 된다.
