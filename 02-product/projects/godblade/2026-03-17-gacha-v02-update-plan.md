## Context

장비뽑기연출기획서가 Ver.0.1 → Ver.0.2로 업데이트되었고, `gacha-client-story-board.md`가 새로 보완되었다. 기존 S3 GDD, S4 개발계획, Q&A 문서, 보완 계획서가 Ver.0.1 기준으로 작성되어 있으므로 v0.2의 변경/확정 사항을 반영해야 한다.

---

## 분석: v0.2에서 변경/확정된 핵심 사항

### 변경 1: 짝 맞추기 이펙트 색상 기준 — 가격대별 → **별(성)별**

**기존 (Q2 확정 기준)**: 가격대(GachaPriceCode)별 이펙트 색상
- 100원: 흰색, 1000원: 파랑, 10000원: 금색

**v0.2 확정**: 장비 별(성) 등급별 RGB 색상 (번개 + 글로우 모두 적용)

| 등급 | RGB | 색상 |
|:----:|-----|------|
| 1~3성 | 255,255,255 | 흰색 |
| 4~6성 | 146,208,80 | 초록 |
| 7~9성 | 0,176,240 | 파랑 |
| 10~11성 | 204,0,0 | 빨강 |

이것은 **가격대 기준이 아닌 장비 등급 기준**으로 확정됨. Q2에 대한 최종 답변이 된다.

### 변경 2: 마을 하단 UI 구성 수정

**기존 S3 GDD (8.0)**:
- 좌측: 상점, 가방, 거래소(신규), 우편함
- 우측: 뽑기(신규), 도전(신규)
- 유지(하단 바 외부): 조력자, 아바타

**v0.2 확정**:
- 좌측: 상점, 가방, 거래소(신규), 우편함, **아바타, 조력자**
- 우측: 뽑기(신규), 도전(신규)
- 제외: 스킬, 보석함, 길드, 연금술, 제작

→ 아바타와 조력자가 좌측 하단 바에 포함됨 (기존: 바 외부 유지)

### 변경 3: 기획서 버전 참조 업데이트

모든 기존 문서의 "Ver.0.1" 참조를 "Ver.0.2"로 갱신.

### 변경 4: Q2 최종 확정

기존 Q2 답변: "가격대별 기준" → v0.2에서 **별(성) 등급별 기준으로 최종 확정**.
가격대별 상자 차등 연출(8.6)은 유지하되, 짝 맞추기 이펙트(번개/글로우) 색상은 별(성) 기준.

### 확인 사항: 기존 반영 내용 (변경 없음)

아래 항목은 v0.1 시점에 이미 기존 문서에 반영 완료되어 v0.2에서도 동일:
- 카드 10장 구성
- 수동/자동(모두 열기) 양방향 조작
- 짝 맞추기 3단계 연출 (번개→강조 회전→결과 확정)
- 버튼 상태 전환 (한번 더 숨김→표시, 모두 열기→상점 가기)
- 꽝 연출 (연기+다음 기회에+닫기)
- 상자 에셋 Fallback
- 뽑기 상점: 기존 상점 UI 재사용
