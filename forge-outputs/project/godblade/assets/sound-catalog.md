### 신규 사운드 추가 시 규칙

- **포맷**: `.wav` (UiFx, Effect) 또는 `.ogg` (UI, 긴 사운드)
- **경로**: 용도에 맞는 하위 폴더에 배치
  - UI 인터랙션 → `Sound/UI/`
  - 특수 효과 → `Sound/UiFx/`
  - 전투/스킬 → `Sound/Effect/`
- **네이밍**: 기존 패턴에 맞춰 `{카테고리}_{설명}.wav`
