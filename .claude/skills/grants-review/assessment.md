# grants-review 스킬 품질 평가 루브릭

> 정부지원사업 검수 스킬의 품질을 5축으로 평가한다.
> 각 축 5점 만점, 총 25점.

## 평가 축

### 1. 검수 완전성 (5점)

5축 검수가 모두 실행되고 결과가 누락 없이 출력되었는가.

| 점수 | 기준 |
|:---:|------|
| 5 | 5축 전체 + 종합점수 + 수정 우선순위 + 리포트 저장 |
| 4 | 5축 전체, 수정 우선순위 또는 저장 중 1건 누락 |
| 3 | 4축 이상 검수, 1축 누락 |
| 2 | 3축 이하 검수 |
| 1 | 형식적 검수만 실행 |

### 2. 팩트체크 정확도 (5점)

본문 수치와 원본 대조가 정확한가.

| 점수 | 기준 |
|:---:|------|
| 5 | 수치 10건+ 대조, 오탈·불일치 정확 탐지, 출처불명 수치 식별 |
| 4 | 수치 7건+ 대조, 대부분 정확 |
| 3 | 수치 5건+ 대조, 일부 누락 |
| 2 | 팩트체크 산발적 |
| 1 | 팩트체크 미실행 |

### 3. 평가 점수 신뢰도 (5점)

평가위원 시뮬레이션의 점수가 실제 평가 결과와 얼마나 유사한가.

| 점수 | 기준 |
|:---:|------|
| 5 | 실제 결과와 ±5점 이내 (사후 검증 시) |
| 4 | 강점/약점 분석이 구체적이고 개선 제안이 실행 가능 |
| 3 | 점수 산출은 되나 근거가 약함 |
| 2 | 점수가 과대/과소 추정 |
| 1 | 점수 산출 미실행 |

**참고**: 실제 평가 결과가 나오기 전에는 강점/약점 분석의 구체성으로 평가.

### 4. 수정 우선순위 유효성 (5점)

제안된 수정 사항이 실제로 점수를 올릴 수 있는가.

| 점수 | 기준 |
|:---:|------|
| 5 | TOP 5 수정 모두 실행 가능 + 예상 점수 상승이 합리적 |
| 4 | TOP 5 중 4건 유효 |
| 3 | TOP 5 중 3건 유효, 일부 모호 |
| 2 | 수정 제안이 추상적이거나 실행 불가 |
| 1 | 수정 우선순위 미생성 |

### 5. 병렬 실행 효율 (5점)

3개 서브에이전트가 올바르게 병렬 스폰되고 결과가 통합되었는가.

| 점수 | 기준 |
|:---:|------|
| 5 | 3개 병렬 스폰, 결과 통합 깔끔, 중복 없음 |
| 4 | 3개 병렬, 경미한 통합 이슈 |
| 3 | 2개만 병렬 또는 순차 실행 |
| 2 | 서브에이전트 미사용 (단일 에이전트로 실행) |
| 1 | 실행 실패 |

## 종합 등급

| 등급 | 점수 범위 | 의미 |
|:---:|:---:|------|
| A+ | 23~25 | 검수 리포트 즉시 활용 가능 |
| A | 20~22 | 경미 보완 후 활용 |
| B+ | 17~19 | 일부 축 재검수 필요 |
| B | 13~16 | 상당 부분 재검수 |
| C | ~12 | 전면 재검수 |
