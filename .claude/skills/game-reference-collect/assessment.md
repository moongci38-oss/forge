# Assessment: game-reference-collect

## 테스트 입력

- input_1: "Collect visual references for a souls-like action RPG combat system"
- input_2: "Gather UI/HUD references from top mobile RPG games"
- input_3: "Collect VFX references for magical spell effects in 2D games"

## 평가 기준 (Yes/No)

1. 레퍼런스 수집 대상(게임명, 장르)이 명시되어 있는가?
2. 수집 방법(WebSearch, 스크린샷, 영상 분석)이 설명되어 있는가?
3. 도구 라우팅(video-reference-guide, screenshot-analyze 등)이 결정되어 있는가?
4. 구조화된 레퍼런스 테이블 또는 목록이 포함되어 있는가?
5. 저장 경로(forge-outputs/01-research/)가 명시되어 있는가?
