# Assessment: product-manager-toolkit

## 테스트 입력

- input_1: "Prioritize these features using RICE framework: dark mode, export PDF, SSO login, mobile app"
- input_2: "Analyze customer interview notes to find common pain points"
- input_3: "Create a PRD for a user notification system"

## 평가 기준 (Yes/No)

1. PM 프레임워크(RICE, MoSCoW 등)가 적용되어 있는가?
2. 정량적 점수 또는 우선순위 순위가 제시되어 있는가?
3. 액션 아이템 또는 다음 단계가 포함되어 있는가?
4. 구조화된 출력(테이블, 섹션 구분)이 있는가?
5. 의사결정 근거 또는 논리가 설명되어 있는가?

## 채점

- 1건 pass = 5개 기준 모두 Yes
- pass_rate = pass 건수 / 전체 실행 수
- 목표: min_pass_rate 0.8 이상
