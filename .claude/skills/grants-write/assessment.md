# grants-write 스킬 품질 평가 루브릭

> 정부지원사업 본문 작성 스킬의 품질을 6축으로 평가한다.
> 각 축 5점 만점, 총 30점.

## 평가 축

### 1. 작성요령 대응률 (5점)

본문이 개요의 작성요령을 얼마나 충실히 반영했는가.

| 점수 | 기준 |
|:---:|------|
| 5 | 모든 작성요령 포인트 1:1 대응, "반드시" 항목 100% 포함, "지양" 항목 0% 포함 |
| 4 | 90% 이상 대응, 경미한 누락 1~2건 |
| 3 | 70% 이상 대응, "반드시" 항목 일부 누락 |
| 2 | 50% 이상 대응, 주요 요구사항 여러 건 누락 |
| 1 | 작성요령과 본문 간 연결이 미약 |

**측정 방법**: 개요의 파란색 [작성요령]을 목록화하고, 본문에서 각 포인트의 반영 여부를 O/X/△로 체크.

### 2. 데이터 활용도 (5점)

분석 보고서의 데이터가 본문에 적절히 인용되었는가.

| 점수 | 기준 |
|:---:|------|
| 5 | 핵심 데이터 TOP 5 전량 인용, 수치 정확, 출처 추적 가능 |
| 4 | TOP 5 중 4건 인용, 수치 정확 |
| 3 | TOP 5 중 3건 인용, 일부 수치 불일치 |
| 2 | 데이터 인용 산발적, 팩트체크 미흡 |
| 1 | 분석 보고서 데이터가 거의 활용되지 않음 |

**측정 방법**: 분석 보고서의 주요 수치를 추출하고, 본문에서 해당 수치의 존재 여부와 정확성을 교차 검증.

### 3. 톤 적절성 (5점)

기획 단계에 맞는 톤으로 작성되었는가.

| 점수 | 기준 |
|:---:|------|
| 5 | 과장/단정/자화자찬 0건, 기획 수준 톤 일관 유지 |
| 4 | 경미한 톤 이탈 1~2건 |
| 3 | 과장 표현 3~5건, 수정 필요 |
| 2 | 단정적/과장적 톤이 빈번 |
| 1 | 톤 관리가 전혀 되지 않음 |

**측정 방법**: 과장 표현 키워드 검색 ("혁신적", "최초의", "초고속", "카피 불가능" 등) + 단정적 문장 패턴 검색.

### 4. 검수자 방향성 반영 (5점)

검수자의 철학·키워드가 본문에 자연스럽게 녹아들었는가.

| 점수 | 기준 |
|:---:|------|
| 5 | 핵심 철학 전량 반영, 금지어 0건, 키워드 자연스러운 배치 |
| 4 | 핵심 철학 대부분 반영, 금지어 0건 |
| 3 | 일부 철학 반영, 금지어 0건 |
| 2 | 금지어 1건 이상 검출 |
| 1 | 검수자 방향성이 무시됨 |

**측정 방법**: 금지어 목록 전문 검색 + 핵심 키워드 출현 빈도 체크.
**비적용 시**: 검수자 방향성이 없으면 이 축은 N/A (나머지 5축으로 평가, 25점 만점).

### 5. 구조 완결성 (5점)

출력 문서의 구조가 완결적인가.

| 점수 | 기준 |
|:---:|------|
| 5 | 용어 설명 + 목차 + 전체 항목 + 개념도 위치 + 색상 컨벤션 모두 준수 |
| 4 | 주요 구조 완비, 경미한 포맷 이슈 1~2건 |
| 3 | 일부 항목 누락 또는 구조 불일치 |
| 2 | 목차와 본문 간 불일치, 색상 컨벤션 미준수 |
| 1 | 구조가 체계적이지 않음 |

**측정 방법**: 용어 테이블, 목차, 섹션 헤더, 파란색/빨간색 span 태그, 개념도 참조 존재 여부 체크.

### 6. 분량 적정성 (5점)

배점 비례 지면 배분이 적절한가.

| 점수 | 기준 |
|:---:|------|
| 5 | 배점 비례 지면 배분 ±10% 이내, 총 분량 페이지 제한 내 |
| 4 | 배점 비례 ±20% 이내, 총 분량 적정 |
| 3 | 일부 항목 과소/과다, 전체 분량은 적정 |
| 2 | 배점과 지면 배분 불균형 심각 |
| 1 | 분량 기준 없이 작성 |

**측정 방법**: 항목별 글자 수 집계 → 배점 비율과 비교.

## 종합 등급

| 등급 | 점수 범위 | 의미 |
|:---:|:---:|------|
| A+ | 28~30 | 검수자 리뷰 직행 가능 |
| A | 25~27 | 경미 수정 후 리뷰 가능 |
| B+ | 22~24 | 부분 재작성 필요 |
| B | 18~21 | 상당 부분 재작성 필요 |
| C | ~17 | 전면 재작성 필요 |

## 기준선 (v1, 2026-03-29)

KOCCA Part2 실전 작성 결과 기준:
- 작성요령 대응률: 4/5 (경미 이슈 2건)
- 데이터 활용도: 4/5 (11건 수정으로 보강 완료)
- 톤 적절성: 4/5 ("초고속 성장팀" 삭제 완료)
- 검수자 방향성: 5/5 (SaaS 금지 준수, Data First 반영)
- 구조 완결성: 5/5 (용어 + 목차 + 11항목 + 개념도 + 색상)
- 분량 적정성: 4/5 (7페이지, 배점 비례 대체로 적정)

**기준선 총점: 26/30 (A등급)**
