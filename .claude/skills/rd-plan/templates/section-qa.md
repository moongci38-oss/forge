# 섹션 QA 루브릭 (3축 60점)

> 서브에이전트(Opus)가 각 섹션 초안을 리뷰. **결함을 찾는 것이 목표.** 관대하지 않는다.

## 평가 대상

- 섹션: {section_id} — {section_title}
- 대응 평가항목: {eval_criteria} ({eval_weight}점)
- 목표 분량: {target_length}자

## A. 콘텐츠 정확도 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 모든 주장에 출처(URL+날짜). 특허/기술 설명이 원본과 일치. 수치 정확. |
| 14-17 | 1-2건 출처 누락. 기술 설명 대체로 정확. |
| 10-13 | 다수 출처 없음. 일부 부정확한 기술 설명. |
| 0-9 | 사실 오류 또는 근거 없는 주장 다수. |

체크리스트:
- [ ] 모든 수치에 출처가 있는가?
- [ ] 특허 내용이 명세서 원본과 일치하는가?
- [ ] 시장 데이터가 검증 가능한 출처인가?
- [ ] 기술 설명에 오류가 없는가?

## B. 평가기준 정렬 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 평가 키워드 정확히 사용. 배점 비중에 맞는 분량. 평가위원 질문 선제 대응. |
| 14-17 | 대부분 키워드 반영. 1-2개 평가 항목 미약. |
| 10-13 | 키워드 사용 불충분. 배점 비중과 분량 불일치. |
| 0-9 | 평가기준과 무관한 서술. |

체크리스트:
- [ ] 이 섹션이 대응하는 평가항목의 키워드를 사용했는가?
- [ ] 분량이 배점 비중에 비례하는가?
- [ ] 평가위원이 "왜?", "어떻게?", "얼마나?"를 물었을 때 답이 있는가?

## C. 작성 품질 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 전문 정부과제 제안서 톤. 논리적 흐름. 격식체 일관. |
| 14-17 | 대체로 전문적. 소수 어색한 표현. |
| 10-13 | 구어체 혼재. 논리 점프. |
| 0-9 | 비전문적. AI 생성 특유 표현. |

체크리스트:
- [ ] 합니다체/입니다체 일관성?
- [ ] "혁신적인", "획기적인" 등 근거 없는 수식어?
- [ ] 주장→근거→시사점 흐름?
- [ ] 다른 섹션과 중복되는 내용?
