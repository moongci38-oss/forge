# 문서 QA 루브릭 (5축 100점)

> 전체 문서 조립 후 최종 품질 검증. **결함을 찾는 것이 목표.**

## A. 섹션 간 일관성 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 전체 문서에서 동일 용어, 수치, 날짜 사용. 섹션 간 모순 없음. |
| 14-17 | 1-2건 용어 불일치 (예: 한 곳에서 "Tag Hub", 다른 곳에서 "태그허브") |
| 10-13 | 수치 불일치 존재 (예: 시장 규모가 섹션마다 다름) |
| 0-9 | 섹션 간 논리적 모순 (예: 1단계 목표가 로드맵과 불일치) |

## B. 평가기준 커버리지 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 모든 평가항목이 최소 1개 섹션에서 정면 대응. 커버리지 매트릭스 100%. |
| 14-17 | 1-2개 평가항목이 간접적으로만 언급 |
| 10-13 | 주요 평가항목 미대응 존재 |
| 0-9 | 다수 평가항목 누락 |

## C. 시각화 품질 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 모든 다이어그램 전문적, 일관된 스타일, 라벨 완비, 해상도 충분 |
| 14-17 | 1-2개 시각화 스타일 불일치 또는 라벨 부족 |
| 10-13 | 텍스트 위주 섹션 다수 (시각화 부족) |
| 0-9 | 시각화 거의 없거나 품질 매우 낮음 |

## D. 분량 준수 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 전체 분량 한도 내. 섹션별 배점 비중에 비례. |
| 14-17 | 전체 분량 ±10% 이내. 1-2개 섹션 비중 불균형. |
| 10-13 | 분량 초과 또는 부족. 다수 섹션 비중 불균형. |
| 0-9 | 심각한 분량 위반. |

## E. 제출 준비도 (20점)

| Score | 기준 |
|:-----:|------|
| 18-20 | 기관 양식 완전 기입. 파일명 규칙. 서명/직인. PDF 변환 오류 없음. |
| 14-17 | 양식 기입 대부분 완료. 1-2개 필드 미기입. |
| 10-13 | 다수 필드 미기입. 파일 형식 오류. |
| 0-9 | 양식 미사용 또는 심각한 제출 오류. |

## 판정

```
Score ≥ 85 → PASS ✅
Score 75-84 → PASS with caveat ⚠️ (이슈 목록 첨부)
Score < 75 → Fix loop (max 3회) → STOP if plateau
```

## Fix Loop

```
문서 QA → Score < 85?
  ↓ Yes
  Top 3 이슈 식별 → 해당 섹션만 재작성 → 재조립 → 재QA
  ↓
  Score ≥ 85? → PASS
  ↓ No
  2차 Fix → 재QA
  ↓
  Score ≥ 85? → PASS
  ↓ No
  3차 Fix → 최종 QA
  ↓
  Score ≥ 75? → PASS with caveat
  Score < 75? → STOP, Human 리뷰 요청
```
