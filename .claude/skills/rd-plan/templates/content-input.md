## 사용자 확인 필수 항목

아래 항목은 자동 감지 후에도 **반드시 사용자 확인**을 받는다:

1. **생성된 맞춤 목차** — "이 목차로 진행할까요?"
2. **분량 배분** — "배점 비중대로 분량을 배분했습니다. 조정이 필요한가요?"
3. **작성 우선순위** — "고배점 섹션부터 작성합니다. 순서 변경이 필요한가요?"
