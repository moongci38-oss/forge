# Assessment: library-search

## 테스트 입력

- input_1: "Search for warrior sprite assets in the Prefab Visual Library"
- input_2: "Find UI button templates matching fantasy RPG style"
- input_3: "Search library for fire/explosion VFX particle effects"

## 평가 기준 (Yes/No)

1. 검색 키워드 또는 태그가 명시되어 있는가?
2. 검색 결과(매칭/부분매칭/없음)가 분류되어 있는가?
3. 매칭된 에셋의 경로 또는 파일명이 제시되어 있는가?
4. MCP 생성 비용 절감 추정이 언급되어 있는가?
5. 다음 단계(재활용/조합/신규 생성) 권장이 포함되어 있는가?

## 채점

- 1건 pass = 5개 기준 모두 Yes
- pass_rate = pass 건수 / 전체 실행 수
- 목표: min_pass_rate 0.8 이상
