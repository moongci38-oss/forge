# Assessment: forge-planning-router

## 테스트 입력

- input_1: "I want to research the market for AI code review tools"
- input_2: "Create a PRD for a new notification feature"
- input_3: "Write a GDD for a mobile puzzle game"

## 평가 기준 (Yes/No)

1. 요청 유형(리서치/PRD/GDD/기획)이 올바르게 식별되어 있는가?
2. 적절한 Forge 스킬 또는 파이프라인이 선택되어 있는가?
3. 라우팅 결정의 근거가 설명되어 있는가?
4. 다음 단계 또는 실행 계획이 안내되어 있는가?
5. Gate 상태(STOP/AUTO-PASS) 또는 Phase가 참조되어 있는가?

## 채점

- 1건 pass = 5개 기준 모두 Yes
- pass_rate = pass 건수 / 전체 실행 수
- 목표: min_pass_rate 0.8 이상
