# AI Content Repurposer — S1 리서치

## 개요

**사업 아이템**: AI 콘텐츠 리퍼포징 SaaS — 긴 형식 콘텐츠(팟캐스트, 유튜브, 웨비나, 블로그)를 멀티플랫폼 포맷(숏폼 영상 스크립트, SNS 포스트, 뉴스레터, 블로그 발췌, 인포그래픽 텍스트)으로 자동 변환하는 도구.

**선정 근거**: 1인 개발자가 내달 1,000만원+(약 $7,500 MRR) 달성 가능성이 가장 높은 아이템. 유사 제품(Castmagic, Repurpose.io)의 시장 검증 완료 + AI 비용 하락으로 마진 개선 + 콘텐츠 크리에이터 시장 폭발적 성장.

**기준 날짜**: 2026-03-30

---

## 경쟁 가설 3개

### 가설 1: AI 콘텐츠 리퍼포징 SaaS
- **포지셔닝**: 콘텐츠 크리에이터(유튜버, 팟캐스터) 대상 올인원 리퍼포징 도구
- **차별화**: Claude API 기반 고품질 한국어/영어 동시 지원 + 플랫폼별 최적화(인스타, 유튜브 숏츠, 트위터, 링크드인)
- **수익 모델**: $29-$79/월 구독 (콘텐츠 처리량 기반 티어)
- **강점**: 보편적 pain point, 높은 반복 사용률, 낮은 이탈률
- **약점**: 경쟁 과열 (Castmagic, Opus Clip, Repurpose.io 등)

### 가설 2: AI 기술 문서 자동화 SaaS
- **포지셔닝**: 개발팀 대상 API 문서/코드 주석/기술 문서 자동 생성
- **차별화**: GitHub/GitLab 연동 + PR 기반 자동 문서 업데이트
- **수익 모델**: $49-$199/월 (repo 수 기반)
- **강점**: forge-workspace.json에 이미 ai-doc-tool로 등록 (기존 구상)
- **약점**: 시장 진입 후 $7,500 MRR까지 시간 소요 예상 (B2B 영업 주기 긴 편)

### 가설 3: AI 마케팅 카피 생성기
- **포지셔닝**: 소규모 이커머스/브랜드 대상 상품 설명, 광고 카피, SEO 콘텐츠 생성
- **차별화**: 한국 이커머스(쿠팡, 네이버 스마트스토어) 특화 + 네이버 SEO 최적화
- **수익 모델**: $19-$49/월 (생성 건수 기반)
- **강점**: 한국 시장 로컬 니치, Leadmore AI $30K MRR 사례
- **약점**: 한국 시장만으로는 TAM 한계, 글로벌 확장 시 차별화 약화

### 핵심 Job: "내 긴 콘텐츠를 모든 플랫폼에 최적화된 형태로 배포하고 싶다"

| Job | 현재 해결 방식 | 불만족 | 우리 솔루션 |
|-----|--------------|--------|-----------|
| 60분 팟캐스트 → 10개 SNS 포스트 | 수동 편집 (4-6시간) | 시간 과다 | AI 자동 추출 + 플랫폼별 포맷팅 (5분) |
| 유튜브 영상 → 블로그 포스트 | 수동 트랜스크립트 편집 | 품질 일관성 부족 | SEO 최적화 블로그 자동 생성 |
| 블로그 → 트위터 쓰레드 | 수동 요약 + 분할 | 톤/길이 맞추기 번거로움 | 플랫폼 톤 자동 적용 |
| 웨비나 → 뉴스레터 | 녹화 다시 보며 정리 | 시간 대비 낮은 ROI | 핵심 포인트 자동 추출 + 뉴스레터 포맷 |

### 감정적 Job
- "나는 콘텐츠 **창작**에 집중하고 싶지, **배포** 작업에 시간을 쓰고 싶지 않다"
- "모든 플랫폼에 존재감을 유지해야 한다는 압박감에서 벗어나고 싶다"

## 선정 근거

**가설 1 (AI 콘텐츠 리퍼포징)을 최종 선정한다.**

| 기준 | 가설 1 | 가설 2 | 가설 3 |
|------|:------:|:------:|:------:|
| 내달 $7,500 MRR 가능성 | ★★★★☆ | ★★☆☆☆ | ★★★☆☆ |
| 1인 개발 가능성 | ★★★★★ | ★★★★☆ | ★★★★★ |
| 시장 검증도 | ★★★★★ | ★★★☆☆ | ★★★★☆ |
| 기술 난이도 (낮을수록 좋음) | ★★★★☆ | ★★★☆☆ | ★★★★★ |
| 글로벌 확장성 | ★★★★★ | ★★★★★ | ★★☆☆☆ |
| **합계** | **21** | **17** | **19** |

**핵심 선정 이유**:
1. **검증된 pain point**: 60분 콘텐츠 리퍼포징에 4-6시간 소요 → 자동화 수요 명확
2. **빠른 수익화**: 콘텐츠 크리에이터는 즉각적 가치를 체감 → 짧은 영업 주기
3. **높은 반복 사용**: 매주/매일 콘텐츠 생산 → 자연스러운 구독 모델
4. **기술 활용**: Claude API + Whisper(트랜스크립션) + Next.js(프론트) — 기존 기술 스택 활용
5. **경쟁 차별화 가능**: 한국어+영어 동시 지원 + 멀티소스 통합이 기존 경쟁사 대비 차별점

## 실행 로드맵 (MVP)

### Phase 1: MVP (4주)
- **Week 1-2**: 핵심 기능 구현
  - 유튜브 URL → 트랜스크립트 추출 (Whisper API)
  - Claude API로 멀티포맷 변환 (트위터 쓰레드, 블로그 요약, 인스타 캡션)
  - 기본 웹 UI (Next.js + 단일 페이지)
- **Week 3**: 결제 + 인증
  - Stripe 구독 결제 ($29/$59/$79 3-tier)
  - 소셜 로그인 (Google/GitHub)
- **Week 4**: 런칭 준비
  - Product Hunt 런칭 준비
  - 랜딩 페이지 + 데모 영상
  - Build in Public 시작 (트위터/인디해커스)

### Phase 2: 성장 (Week 5-8)
- 팟캐스트 RSS 피드 연동
- 블로그 URL 입력 지원
- 플랫폼별 톤/길이 커스터마이징
- SEO 최적화 블로그 자동 생성

### Phase 3: 확장 (Week 9-12)
- 자동 SNS 발행 연동 (Buffer API)
- 뉴스레터 자동 생성 (Mailchimp/Substack 연동)
- 한국어 특화 기능 (네이버 블로그, 카카오 채널)
- 팀 기능 (다중 사용자)

## 기술 스택 제안

| 레이어 | 기술 | 이유 |
|--------|------|------|
| 프론트엔드 | Next.js 16.2 + TypeScript | 기존 Portfolio 스택 활용, AI-ready 스캐폴딩 |
| 백엔드 | Next.js API Routes + FastAPI | 간단한 로직은 API Routes, ML 파이프라인은 FastAPI |
| AI | Claude API (Sonnet 4.6) | 텍스트 생성 품질 + 한국어 지원 |
| 트랜스크립션 | Whisper API / Deepgram | 오디오→텍스트 변환 |
| DB | PostgreSQL + Prisma | 구독 관리, 사용자 데이터 |
| 결제 | Stripe | 글로벌 구독 결제 |
| 인프라 | Vercel (프론트) + Railway (백엔드) | 솔로 개발자 운영 최적화 |
| 모니터링 | Sentry + PostHog | 에러 추적 + 사용자 분석 |
