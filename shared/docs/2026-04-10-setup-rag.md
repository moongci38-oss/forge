# RAG 검색 설치 가이드

> `/rag-search` 스킬용 하이브리드 벡터+BM25 시맨틱 검색 엔진
> 소요 시간: ~5분 (최초 인덱스 빌드 제외)

---

## 사전 요구사항

- Python 3.10+
- `~/forge/.env`에 `OPENAI_API_KEY` 설정 완료
- forge-outputs 디렉토리에 문서 존재

---

## 설치

### 1. Python 패키지 설치

```bash
pip install -r ~/forge/shared/scripts/rag/requirements.txt
```

설치되는 주요 패키지:

| 패키지 | 용도 |
|--------|------|
| `llama-index-core` | RAG 파이프라인 코어 |
| `llama-index-embeddings-openai` | OpenAI 임베딩 (text-embedding-3-small) |
| `llama-index-vector-stores-faiss` | 벡터 인덱스 저장소 |
| `llama-index-retrievers-bm25` | BM25 키워드 검색 |
| `faiss-cpu` | 고속 벡터 유사도 검색 |
| `python-docx`, `pymupdf`, `python-pptx`, `openpyxl` | 문서 파서 |
| `pyhwp` | HWP 텍스트 추출 |
| `pytesseract` | 이미지 OCR |

### 2. (선택) OCR 지원 — 스캔 이미지 포함 문서용

```bash
sudo apt install tesseract-ocr tesseract-ocr-kor
```

### 3. 인덱스 빌드

```bash
# 정부과제 문서 인덱스 (기본)
bash ~/forge/shared/scripts/rag/setup.sh ~/forge-outputs/09-grants

# 전체 forge-outputs 인덱스 (선택)
bash ~/forge/shared/scripts/rag/setup.sh ~/forge-outputs
```

최초 실행 시 문서 파싱 + 임베딩 생성에 수 분 소요. 이후 실행은 증분 업데이트.

## 사용법

### Claude Code에서

```
/rag-search 플랫폼 프랜차이즈 전략
/rag-search TagHub 기술 차별점
/rag-search 시장 규모 전망
```

### CLI에서 직접

```bash
# 기본 검색
python3 ~/forge/shared/scripts/rag/search.py "투자 유치 전략"

# 결과 수 조정
python3 ~/forge/shared/scripts/rag/search.py "PoC 검증" --top-k 10

# 검색 모드 지정
python3 ~/forge/shared/scripts/rag/search.py "시장 규모" --mode vector
python3 ~/forge/shared/scripts/rag/search.py "시장 규모" --mode bm25

# JSON 출력
python3 ~/forge/shared/scripts/rag/search.py "기술 스택" --json
```

## 인덱스 업데이트

forge-outputs에 문서가 추가/변경되면 재빌드:

```bash
bash ~/forge/shared/scripts/rag/setup.sh ~/forge-outputs/09-grants
```

## 트러블슈팅

| 문제 | 해결 |
|------|------|
| `OPENAI_API_KEY 미설정` | `~/forge/.env`에 `OPENAI_API_KEY=sk-...` 추가 |
| `faiss 설치 실패` | `pip install faiss-cpu --no-cache-dir` 재시도 |
| `pyhwp 설치 실패` | HWP 지원 제외하고 나머지 패키지만 설치 가능 |
| `tesseract not found` | `sudo apt install tesseract-ocr` 실행 |
| 검색 결과 0건 | 인덱스 빌드 완료 여부 확인. `setup.sh` 재실행 |
| 느린 검색 | `--top-k` 줄이기 또는 `faiss-gpu` 설치 (CUDA 환경) |
